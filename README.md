# JAM Skills for Claude Code

AI-powered distribution intelligence from [JAM](https://spreadjam.com).

## Installation

```bash
/plugin add https://github.com/wespreadjam/jam-skills
```

## /geo-optimize

Optimize your content to get cited by AI search engines (ChatGPT, Perplexity, Claude, Google AI Overview).

```bash
/geo-optimize                                    # Analyze current codebase
/geo-optimize "saas pricing"                     # Analyze for specific keyword
/geo-optimize https://example.com "keyword"      # Analyze deployed URL
```

**What it does:**
- Fetches keyword intelligence (search volume, difficulty, intent)
- Analyzes competitors ranking for your keyword
- Applies proven GEO tactics (FAQ schema, comparison tables, answer capsules)
- Edits your code directly with optimizations

**GEO tactics included:**
| Tactic | Citation Improvement |
|--------|---------------------|
| FAQ Schema | +156% |
| Comparison Tables | +112% |
| Statistics with Sources | +40% |
